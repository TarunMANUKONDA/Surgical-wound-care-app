# Surgical Wound Care App

## Project Structure

- `Surgical-wound-care-app/`: Contains the React frontend code.
- `backend-files/`: Contains the PHP backend code.

## How to Run the Frontend

The frontend code is located in the `Surgical-wound-care-app` subdirectory.

### Option 1: Use the helper script
Double-click `start_frontend.bat` in this directory.

### Option 2: Manual startup
1. Open a terminal in this directory.
2. Run `cd Surgical-wound-care-app`
3. Run `npm run dev`

## Backend Setup
Follow the instructions in `backend-files/xampp_setup_guide.md`.
